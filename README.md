# Поймай прокойны

Демо-версия мини-игры для вертикального планшета на `React + TypeScript + Vite`.

## Запуск

```bash
npm install
npm run dev
```

Приложение открывается локально через Vite. Dev-сервер настроен с `host: true`, поэтому игру можно открыть и с другого устройства в одной сети по IP вашего компьютера.

## Что реализовано

- случайная генерация плавного `SVG path` маршрута для каждого нового раунда
- движение бегунка по пути через `requestAnimationFrame`
- 4 призовые зоны, привязанные к прогрессу вдоль маршрута
- отдельный модуль расчёта попадания и очков
- адаптация под вертикальный fullscreen-экран
